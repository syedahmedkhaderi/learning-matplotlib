# Learning Matplotlib

A collection of Jupyter notebooks and resources for learning data visualization with Matplotlib.

## Repository Structure

This repository is organized by visualization types:

- **01-plot**: Basic line plots and formatting.
- **02-bar**: Categorical data with bar charts.
- **03-scatter**: Relationship between variables using scatter plots.
- **04-piecharts**: Proportional data with pie charts.
- **05-histogram**: Data distribution through histograms.
- **06-box-plot**: Statistical summaries with box plots.
- **07-stack**: Cumulative data over time or categories.
- **08-sub-plots**: Multiple visualizations in a single figure.

## Resources

- `Data Visualization - Matplotlib.pdf`: A comprehensive guide on Matplotlib.
- `morden-matplotlib.ipynb`: Additional modern Matplotlib techniques.

## Getting Started

1. Clone this repository.
2. Install the necessary dependencies (Jupyter, Matplotlib, Numpy).
3. Open the notebooks in Jupyter Lab or Jupyter Notebook.
